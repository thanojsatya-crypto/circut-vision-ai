### 1.2 Semiconductor Equilibrium and Transport

| Formula | Meaning |
|---|---|
| \(\displaystyle V_T=\frac{kT}{q}\) | Thermal voltage |
| \(\displaystyle n=N_C e^{-(E_C-E_F)/kT}\) | Electron concentration in conduction band |
| \(\displaystyle p=N_V e^{-(E_F-E_V)/kT}\) | Hole concentration in valence band |
| \(\displaystyle n_i=\sqrt{N_CN_V}\,e^{-E_g/(2kT)}\) | Intrinsic carrier concentration |
| \(\displaystyle np=n_i^2\) | Mass-action law at thermal equilibrium |
| \(\displaystyle E_i=\frac{E_C+E_V}{2}+\frac{kT}{2}\ln\!\left(\frac{N_V}{N_C}\right)\) | Intrinsic Fermi level |
| \(\displaystyle n=n_i e^{(E_{Fn}-E_i)/kT}\) | Electron concentration using electron quasi-Fermi level |
| \(\displaystyle p=n_i e^{(E_i-E_{Fp})/kT}\) | Hole concentration using hole quasi-Fermi level |
| \(\displaystyle E_{Fn}-E_{Fp}=qV_A\) | Quasi-Fermi level separation under applied junction voltage \(V_A\) |
| \(\displaystyle p+N_D^+=n+N_A^-\) | Charge neutrality condition |
| \(\displaystyle n_0\approx N_D,\quad p_0\approx \frac{n_i^2}{N_D}\) | Nondegenerate n-type semiconductor |
| \(\displaystyle p_0\approx N_A,\quad n_0\approx \frac{n_i^2}{N_A}\) | Nondegenerate p-type semiconductor |
| \(\displaystyle E_C-E_F\approx kT\ln\!\left(\frac{N_C}{N_D}\right)\) | Fermi level below conduction band in n-type material |
| \(\displaystyle E_F-E_V\approx kT\ln\!\left(\frac{N_V}{N_A}\right)\) | Fermi level above valence band in p-type material |
| \(\displaystyle \sigma=q(n\mu_n+p\mu_p)\) | Semiconductor conductivity |
| \(\displaystyle \rho=\frac{1}{\sigma}\) | Resistivity |
| \(\displaystyle R=\rho\frac{L}{A}\) | Resistance of semiconductor sample |
| \(\displaystyle J_{n,\text{drift}}=qn\mu_nE\) | Electron drift current density |
| \(\displaystyle J_{p,\text{drift}}=qp\mu_pE\) | Hole drift current density |
| \(\displaystyle J_{n,\text{diff}}=qD_n\frac{dn}{dx}\) | Electron diffusion current density |
| \(\displaystyle J_{p,\text{diff}}=-qD_p\frac{dp}{dx}\) | Hole diffusion current density |
| \(\displaystyle \frac{D_n}{\mu_n}=\frac{D_p}{\mu_p}=\frac{kT}{q}=V_T\) | Einstein relation |
| \(\displaystyle J_n=J_{n,\text{drift}}+J_{n,\text{diff}}\) | Total electron current density |
| \(\displaystyle J_p=J_{p,\text{drift}}+J_{p,\text{diff}}\) | Total hole current density |
| \(\displaystyle \frac{\partial n}{\partial t}=\frac{1}{q}\frac{\partial J_n}{\partial x}+G_n-R_n\) | Electron continuity equation |
| \(\displaystyle \frac{\partial p}{\partial t}=-\frac{1}{q}\frac{\partial J_p}{\partial x}+G_p-R_p\) | Hole continuity equation |
| \(\displaystyle L_n=\sqrt{D_n\tau_n}\) | Electron diffusion length |
| \(\displaystyle L_p=\sqrt{D_p\tau_p}\) | Hole diffusion length |
| \(\displaystyle \Delta n(x)=\Delta n(0)e^{-x/L_n}\) | Excess electron decay in steady state |
| \(\displaystyle \Delta p(x)=\Delta p(0)e^{-x/L_p}\) | Excess hole decay in steady state |
| \(\displaystyle R_H\approx-\frac{1}{qn}\) | Hall coefficient for n-type material |
| \(\displaystyle R_H\approx+\frac{1}{qp}\) | Hall coefficient for p-type material |
| \(\displaystyle V_H=R_H\frac{IB}{t}\) | Hall voltage |
| \(\displaystyle E_g(T)=E_g(0)-\frac{\alpha T^2}{T+\beta}\) | Varshni bandgap temperature relation |
| \(\displaystyle n_i(T)\propto T^{3/2}e^{-E_g/(2kT)}\) | Approximate intrinsic concentration temperature dependence |

## 2. DESIGN ORDER OF OPERATIONS

### 2.1 General Semiconductor Device Analysis Flow

1. Identify semiconductor material: Si, Ge, GaAs, etc.
2. Identify temperature \(T\).
3. Identify doping: \(N_D\), \(N_A\).
4. Calculate intrinsic concentration \(n_i\).
5. Calculate majority and minority carrier concentrations.
6. Calculate conductivity \(\sigma=q(n\mu_n+p\mu_p)\).
7. Calculate resistivity \(\rho=1/\sigma\).
8. Calculate diffusion coefficients using \(D/\mu=V_T\).
9. Calculate diffusion lengths \(L=\sqrt{D\tau}\).
10. Apply bias condition: forward, reverse, zero bias.
11. Calculate depletion width, electric field, capacitance, or current.
12. Check power dissipation and breakdown limits.
13. Check thermal limits using \(T_J=T_A+P_DR_{\theta JA}\).

---

### 2.2 PN Junction Diode Design Flow

1. Specify required forward current \(I_F\).
2. Select diode type: signal, rectifier, Schottky, Zener, LED.
3. Estimate forward voltage \(V_D\).
4. Calculate series resistance:

\[
R_S=\frac{V_S-V_D}{I_F}
\]

5. Calculate resistor power:

\[
P_R=I_F^2R_S
\]

6. Calculate diode power:

\[
P_D=V_DI_F
\]

7. Select diode ratings:
   - \(I_F\) rating \(>\) operating current.
   - PIV rating \(>\) maximum reverse voltage.
   - Power rating \(>\) calculated dissipation.
8. Check reverse leakage and breakdown.
9. Recheck at temperature extremes.

---

### 2.3 Rectifier Power-Supply Design Flow

1. Specify required DC output voltage \(V_{DC}\).
2. Specify load current \(I_L\).
3. Specify maximum ripple \(V_r(pp)\).
4. Select rectifier topology:
   - Half-wave: simple, high ripple.
   - Full-wave center-tap: lower ripple, transformer center tap needed.
   - Bridge: most common, two diode drops.
5. Determine ripple frequency:
   - Half-wave: \(f_r=f\)
   - Full-wave/bridge: \(f_r=2f\)
6. Calculate filter capacitor:

\[
C=\frac{I_L}{f_rV_r(pp)}
\]

7. Calculate required secondary peak voltage:

\[
V_{sec,pk}=V_{DC}+\frac{V_r(pp)}{2}+n_DV_D
\]

8. Calculate RMS secondary voltage:

\[
V_{sec,rms}=\frac{V_{sec,pk}}{\sqrt2}
\]

9. Select diode PIV rating:
   - Half-wave: \(>V_m\)
   - Center-tap: \(>2V_m\)
   - Bridge: \(>V_m\)
10. Select diode current rating:
   - Average diode current \(>\) calculated average.
   - Surge current \(>\) capacitor charging surge.
11. Select transformer VA rating:

\[
VA\approx V_{sec,rms}I_{sec,rms}
\]

12. Add fuse, bleeder resistor, and voltage regulator if required.

---

### 2.4 Zener Voltage Regulator Design Flow

1. Specify output voltage \(V_Z\).
2. Specify load current range:
   - \(I_{L,min}\)
   - \(I_{L,max}\)
3. Specify input voltage range:
   - \(V_{S,min}\)
   - \(V_{S,max}\)
4. Select Zener power rating \(P_Z\).
5. Calculate maximum Zener current:

\[
I_{Z,max}=\frac{P_Z}{V_Z}
\]

6. Select minimum Zener current:
   - Small Zener: \(I_{Z,min}\approx1\text{ to }5\ \mathrm{mA}\)
   - Power Zener: \(I_{Z,min}\approx10\text{ to }20\ \mathrm{mA}\)
7. Calculate maximum series resistor:

\[
R_{S,max}=\frac{V_{S,min}-V_Z}{I_{Z,min}+I_{L,max}}
\]

8. Calculate minimum series resistor:

\[
R_{S,min}=\frac{V_{S,max}-V_Z}{I_{Z,max}+I_{L,min}}
\]

9. Choose standard resistor:

\[
R_{S,min}<R_S<R_{S,max}
\]

10. Calculate resistor power:

\[
P_R=(I_Z+I_L)^2R_S
\]

11. Verify regulation and Zener dissipation at all input/load extremes.

---

### 2.5 LED Driver Design Flow

1. Specify LED forward current \(I_F\).
2. Obtain LED forward voltage \(V_F\) from datasheet.
3. Select supply voltage \(V_S\).
4. Calculate resistor:

\[
R=\frac{V_S-V_F}{I_F}
\]

5. Calculate resistor power:

\[
P_R=I_F^2R
\]

6. Select resistor with at least \(2\times\) power margin.
7. Verify LED power:

\[
P_{LED}=V_FI_F
\]

8. For high-current LEDs, use constant-current driver instead of resistor-only drive.

---

### 2.6 BJT Switch Design Flow

1. Specify load current \(I_C\).
2. Specify supply voltage \(V_{CC}\).
3. Select BJT with:
   - \(I_C\) rating \(>\) load current.
   - \(V_{CEO}\) rating \(>\) supply voltage.
   - adequate power rating.
4. Choose forced beta:

\[
\beta_{forced}\approx10
\]

5. Calculate required base current:

\[
I_B=\frac{I_C}{\beta_{forced}}
\]

6. Calculate base resistor:

\[
R_B=\frac{V_{drive}-V_{BE}}{I_B}
\]

7. Check saturation:

\[
V_{CE(sat)}<V_{load,allowed}
\]

8. Calculate base-drive power and resistor power.
9. Add flyback diode for inductive loads.
10. Add base resistor and optional base-emitter pull-down resistor.

---

### 2.7 BJT Common-Emitter Amplifier Design Flow

1. Select supply voltage \(V_{CC}\).
2. Choose quiescent collector current \(I_C\).
3. Set emitter voltage:

\[
V_E\approx0.1V_{CC}\text{ to }0.2V_{CC}
\]

4. Calculate emitter resistor:

\[
R_E=\frac{V_E}{I_E}\approx\frac{V_E}{I_C}
\]

5. Set collector voltage near mid-supply:

\[
V_C\approx\frac{V_{CC}}{2}
\]

6. Calculate collector resistor:

\[
R_C=\frac{V_{CC}-V_C}{I_C}
\]

7. Calculate \(V_{CEQ}\):

\[
V_{CEQ}=V_C-V_E
\]

8. Select voltage-divider current:

\[
I_{divider}\ge10I_B
\]

9. Calculate \(R_1,R_2\) from \(V_B=V_E+V_{BE}\).
10. Calculate small-signal parameters:
   - \(g_m=I_C/V_T\)
   - \(r_e=V_T/I_E\)
   - \(r_\pi=\beta/g_m\)
11. Calculate gain:

\[
A_v\approx-\frac{R_C\parallel R_L}{r_e+R_E}
\]

12. Select coupling capacitors so:

\[
X_C\ll R_{in}\quad\text{at lowest signal frequency}
\]

13. Select emitter bypass capacitor so:

\[
X_{C_E}\ll R_E\quad\text{at lowest signal frequency}
\]

14. Verify output swing without cutoff or saturation clipping.

---

### 2.8 JFET Bias Design Flow

1. Obtain \(I_{DSS}\) and \(V_P\) from datasheet.
2. Choose desired drain current \(I_D\).
3. Calculate required \(V_{GS}\):

\[
V_{GS}=V_P\left(1-\sqrt{\frac{I_D}{I_{DSS}}}\right)
\]

4. Select source resistor for self-bias:

\[
R_S=-\frac{V_{GS}}{I_D}
\]

5. Select drain resistor for desired \(V_D\):

\[
R_D=\frac{V_{DD}-V_D}{I_D}
\]

6. Verify saturation:

\[
V_{DS}\ge V_{GS}-V_P
\]

7. Calculate transconductance:

\[
g_m=\frac{2I_{DSS}}{|V_P|}\left(1-\frac{V_{GS}}{V_P}\right)
\]

8. Calculate voltage gain:

\[
A_v\approx-g_m(R_D\parallel R_L)
\]

---

### 2.9 MOSFET Switch Design Flow

1. Specify load voltage and load current.
2. Select N-channel MOSFET for low-side switching or P-channel MOSFET for high-side switching.
3. Verify \(V_{DS,max}>V_{supply}\).
4. Verify \(I_D\) rating \(>\) load current.
5. Verify \(R_{DS(on)}\) at available gate voltage.
6. Calculate conduction loss:

\[
P_D=I_D^2R_{DS(on)}
\]

7. Check junction temperature:

\[
T_J=T_A+P_DR_{\theta JA}
\]

8. Select gate-drive voltage:
   - Logic-level MOSFET: fully enhanced at \(3.3\ \mathrm{V}\) or \(5\ \mathrm{V}\)
   - Standard MOSFET: usually requires \(10\ \mathrm{V}\)
9. Add gate resistor:
   - \(10\ \Omega\) to \(100\ \Omega\) typical
10. Add gate-source pull-down resistor:
   - \(10\ \mathrm{k\Omega}\) to \(100\ \mathrm{k\Omega}\) typical
11. Add flyback diode for inductive loads.
12. Add snubber or TVS if switching inductive or high-voltage loads.

---

### 2.10 MOSFET Amplifier Design Flow

1. Select MOSFET and obtain \(k_n\), \(V_T\), \(\lambda\).
2. Choose drain current \(I_D\).
3. Calculate required overdrive:

\[
V_{OV}=\sqrt{\frac{2I_D}{k_n}}
\]

4. Calculate gate-source voltage:

\[
V_{GS}=V_T+V_{OV}
\]

5. Select drain resistor or active load.
6. Verify saturation:

\[
V_{DS}\ge V_{OV}
\]

7. Calculate transconductance:

\[
g_m=\sqrt{2k_nI_D}
\]

8. Calculate gain:

\[
A_v\approx-g_m(R_D\parallel r_o\parallel R_L)
\]

9. Check output swing.
10. Add coupling and bypass capacitors for desired bandwidth.

---

### 2.11 Photodiode Design Flow

1. Specify illumination level or required photocurrent.
2. Choose photodiode responsivity \(R_\lambda\).
3. Estimate photocurrent:

\[
I_{ph}=R_\lambda P_{opt}
\]

4. Choose mode:
   - Photoconductive mode: reverse bias, faster response.
   - Photovoltaic mode: zero bias, lower noise.
5. Select load or transimpedance resistor:

\[
V_{out}=I_{ph}R_f
\]

6. Check bandwidth:

\[
f_{-3dB}\approx\frac{1}{2\pi R_fC_{total}}
\]

7. Check dark current and shot noise.

---

### 2.12 Laboratory Characterization Flow

| Device | Measurement Flow |
|---|---|
| Diode I-V | Set current-limiting resistor → sweep voltage → measure \(V_D,I_D\) → plot I-V → extract \(n\) and \(I_S\) |
| Zener regulator | Vary \(V_S\) → measure \(V_Z,I_Z\) → calculate line regulation → vary \(I_L\) → calculate load regulation |
| Rectifier | Measure transformer RMS → measure no-load DC → connect load → measure ripple → calculate \(C\) and ripple factor |
| BJT output characteristics | Set fixed \(I_B\) → sweep \(V_{CE}\) → measure \(I_C\) → extract \(\beta\) and Early voltage |
| BJT transfer characteristic | Sweep \(V_{BE}\) → measure \(I_C\) → verify exponential region |
| JFET characteristics | Sweep \(V_{GS}\) → measure \(I_D\) → extract \(I_{DSS}\) and \(V_P\) |
| MOSFET transfer characteristic | Sweep \(V_{GS}\) → measure \(I_D\) → extract \(V_T\) and \(k_n\) |
| MOSFET output characteristic | Fix \(V_{GS}\) → sweep \(V_{DS}\) → identify triode and saturation regions |

## 4. PRACTICAL COMPONENT SELECTION RULES

### 4.1 Resistors

| Selection Rule | Use |
|---|---|
| Use metal-film resistors | General lab circuits, low noise, better tolerance |
| Use carbon-film resistors | Low-cost general-purpose circuits |
| Use wirewound resistors | High-power loads, current sensing |
| Use \(1\%\) tolerance | Bias networks, feedback, measurement circuits |
| Use \(5\%\) tolerance | LED resistors, pull-ups, noncritical circuits |
| Select power rating \(\ge2P_{calc}\) | Thermal safety margin |
| Avoid using potentiometer as high-current rheostat without rating check | Prevent overheating |
| Use pull-up/pull-down resistors | Prevent floating MOSFET gates and logic inputs |

---

### 4.2 Capacitors

| Selection Rule | Use |
|---|---|
| Use ceramic capacitors | Decoupling, bypass, high-frequency filtering |
| Use electrolytic capacitors | Bulk filtering in power supplies |
| Use film capacitors | Audio coupling, precision timing, AC applications |
| Use tantalum capacitors carefully | Compact bulk capacitance; observe polarity and derating |
| Always observe polarity for electrolytic/tantalum capacitors | Reverse polarity causes leakage, heating, failure |
| Place \(100\ \mathrm{nF}\) ceramic close to IC supply pins | High-frequency decoupling |
| Place \(10\ \mu\mathrm{F}\) electrolytic near regulator input/output | Low-frequency bulk decoupling |
| Use voltage derating | Prevent dielectric breakdown and capacitance loss |
| Do not use electrolytic capacitors in high-frequency RF paths | High ESR/ESL |

### 4.5 MOSFETs

## 5. COMMON MISTAKES & TROUBLESHOOTING

| # | Mistake | Symptoms | Corrective Action |
|---:|---|---|---|
| 1 | Diode, LED, electrolytic capacitor, or transistor polarity reversed | No output, LED not glowing, overheating, smoke, blown fuse | Check datasheet pinout, use multimeter diode mode, verify polarity before power |
| 2 | LED or Zener used without current limiting | LED burns out, Zener overheats, supply current high | Add series resistor; calculate \(R=(V_S-V_F)/I_F\) for LED or \(R_S=(V_S-V_Z)/(I_Z+I_L)\) for Zener |
| 3 | BJT switch not saturating | Load under-driven, transistor hot, \(V_{CE}\) too high | Increase base current; use forced beta \(10\text{ to }20\); recalculate \(R_B\) |
| 4 | MOSFET not fully on | MOSFET hot, load voltage low, high \(R_{DS(on)}\) behavior | Use logic-level MOSFET; ensure \(V_{GS}\) meets datasheet condition; add gate pull-down |
| 5 | Rectifier/filter designed with wrong PIV or ripple frequency | Diode failure, excessive ripple, output voltage lower than expected | Recalculate PIV; use \(f_r=f\) for half-wave and \(f_r=2f\) for full-wave/bridge; increase capacitor value |

## 6. VIVA & INTERVIEW TRAPS

| # | Question | Correct Answer |
|---:|---|---|
| 1 | Why does a forward-biased silicon diode often appear to have a fixed \(0.7\ \mathrm{V}\) drop if the diode equation is exponential? | The diode current depends exponentially on voltage, so a large current change requires only a small voltage change. Around normal operating current, \(V_D\) varies slowly and is approximated as \(0.7\ \mathrm{V}\). |
| 2 | If a pn junction has a built-in potential \(V_{bi}\), why can it not be measured directly using a voltmeter across an unbiased diode? | Metal-semiconductor contact potentials cancel the internal junction potential in an open circuit. At equilibrium, net current is zero, so no external voltage appears across the diode terminals. |
| 3 | Is a BJT truly current-controlled? | In circuit models, \(I_C=\beta I_B\) is useful, but physically the collector current is primarily controlled by \(V_{BE}\). Base current exists mainly because of recombination and injection requirements. |
