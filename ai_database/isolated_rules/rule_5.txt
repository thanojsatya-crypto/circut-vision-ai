## 1. CORE FORMULAS & MATHEMATICS
| Category | Formula | Description |
|----------|---------|-------------|
| **Power** | \(P = V \times I\) | Instantaneous power |
| | \(P_{avg} = V_{rms} \times I_{rms}\) | Average power for AC |
| | \(P_{loss}=I^{2}R\) | Conduction loss in conductors |
| | \(P_{switch}= \frac{1}{2} V_{DS} I_{D} t_{on/off} f_{sw}\) | Switching loss in MOSFET/IGBT |
| **Efficiency** | \(\eta = \frac{P_{out}}{P_{in}} \times 100\%\) | Overall efficiency |
| | \(\eta_{conv}= \frac{V_{out} I_{out}}{V_{in} I_{in}}\) | Converter efficiency |
| **Voltage Regulation** | \(\Delta V_{out}=V_{out, no‑load} - V_{out, full‑load}\) | Regulation spec |
| **Ripple** | \(V_{ripple}= \frac{I_{out}}{f_{sw} C}\) (buck/boost) | Output voltage ripple |
| **Inductor Energy** | \(E_L = \frac{1}{2} L I^{2}\) | Energy stored in inductor |
| **Capacitor Ripple Current** | \(I_{C(ripple)} = C \times \frac{dV}{dt}\) | RMS ripple current rating |
| **Duty Cycle** | \(D = \frac{V_{out}}{V_{in}}\) (ideal buck) | Duty ratio |
| | \(D = \frac{V_{in}}{V_{out}}\) (ideal boost) | Duty ratio |
| **Switching Frequency** | \(f_{sw}= \frac{1}{T_{on}+T_{off}}\) | Frequency of PWM |
| **Gain (Linear Regulator)** | \(A_v = 1 + \frac{R_f}{R_g}\) | Non‑inverting op‑amp gain (used in error amp) |
| **Loop Bandwidth** | \(f_{BW}= \frac{1}{2\pi R_{c} C_{c}}\) | Compensation network bandwidth |
| **Thermal** | \(\Delta T = P_{loss} \times \theta_{JA}\) | Junction‑to‑ambient temperature rise |
| **Safety** | \(P_{max}= \frac{V_{max}}{R_{min}}\) | Max power allowed by component rating |

## 2. DESIGN PROCEDURE (Step‑by‑Step)
1. **Define specifications**  
   - Input voltage range (min, max, nominal)  
   - Output voltage(s) & tolerance  
   - Output current & power  
   - Efficiency target, ripple, transient load step, standby power, operating temperature.  
2. **Select topology**  
   - Buck, boost, buck‑boost, SEPIC, flyback, forward, linear regulator, etc., based on voltage conversion ratio and isolation needs.  
3. **Calculate duty cycle & switching frequency**  
   - Use ideal duty formulas, then apply margin for component limits.  
4. **Determine inductor value**  
   - \(L = \frac{(V_{in(min)}-V_{out}) \times D}{\Delta I_{L} \times f_{sw}}\) (buck)  
   - Choose \(\Delta I_{L}\) ≈ 20‑40 % of I_out.  
5. **Select output capacitor(s)**  
   - \(C_{out} \ge \frac{I_{out} \times D}{V_{ripple} \times f_{sw}}\)  
   - Verify ESR and ripple‑current rating.  
6. **Choose input filter** (LC or π) to meet EMI limits.  
7. **Select switching device** (MOSFET/IGBT)  
   - V_DS ≥ 1.5 × V_in(max)  
   - I_D ≥ 2 × I_out (including ripple).  
   - R_DS(on) low for efficiency.  
8. **Select diode (if non‑synchronous)**  
   - Reverse voltage ≥ V_in(max)  
   - Forward current ≥ I_out (peak).  
9. **Design control loop**  
   - Pick error‑amp, compensation network (R‑C).  
   - Target phase margin 45‑60°.  
   - Simulate Bode plot.  
10. **Thermal analysis**  
    - Compute total loss (conduction + switching + quiescent).  
    - Choose heatsink or copper area to keep \(T_j < T_{j,max}\).  
11. **Component derating**  
    - Apply 20 % voltage derating, 50 % current derating for safety.  
12. **Create schematic & PCB layout**  
    - Minimize loop area for high‑frequency paths (switch‑MOSFET‑inductor‑output).  
    - Place input and output caps close to IC pins.  
    - Use wide copper pours for ground and power.  
13. **Prototype on breadboard or evaluation board** (if possible)  
    - Verify start‑up, load regulation, line regulation, transient response.  
14. **Iterate**  
    - Adjust component values, compensation, or layout based on measurements.  
15. **Finalize BOM**  
    - List part numbers, footprints, ratings, and supplier sources.

## 5. BREADBOARD & PCB TROUBLESHOOTING
| # | Symptom | Probable Cause | Test Method | Fix |
|---|----------|----------------|-------------|-----|
| 1 | No output voltage | Power‑on reset pin floating / EN pin low | Measure EN pin voltage; check pull‑up resistor | Connect EN to VIN via proper resistor (e.g., 10 kΩ) |
| 2 | Excessive ripple (>5 % of Vout) | Output capacitor ESR too high or insufficient C | Measure ripple with oscilloscope; check ESR with LCR meter | Replace with low‑ESR MLCC + bulk electrolytic; add parallel ceramic |
| 3 | Over‑temperature of MOSFET | High switching frequency + large R_DS(on) + inadequate heatsink | Infrared camera or thermistor on MOSFET die | Increase gate drive voltage, add heatsink, or select lower R_DS(on) MOSFET |
| 4 | Oscillation / unstable regulation | Loop compensation inadequate (phase margin <45°) | Bode plot using network analyzer or probe the error amp | Redesign compensation network (adjust R_C, C_C values) |
| 5 | Output voltage drops under load step | Inductor too small → high ΔI causing droop | Measure inductor current ripple; calculate L value | Increase inductance or reduce switching frequency; add output capacitor for better transient response |

## 6. VIVA & INTERVIEW TRAPS
| Q# | Question | Correct Answer & Brief Explanation |
|----|----------|--------------------------------------|
| 1 | *Why does a synchronous buck converter achieve higher efficiency than a diode‑based buck?* | The synchronous MOSFET replaces the Schottky diode, eliminating the diode’s forward voltage drop (≈0.4‑0.6 V). At high output currents the loss saved (I·V_f) is substantial, raising efficiency. |
| 2 | *Explain the impact of duty‑cycle variation on inductor current ripple in a boost converter.* | In boost, \(ΔI_L = \frac{V_{in} D}{L f_{sw}}\). As D increases (higher output voltage), the on‑time grows, raising ΔI_L linearly. Larger ripple can cause core saturation and higher EMI. |
| 3 | *What is the purpose of the “dead‑time” in a half‑bridge driver, and how does it affect switching losses?* | Dead‑time prevents shoot‑through (both MOSFETs on simultaneously). Too little dead‑time causes cross‑conduction → catastrophic failure; too much dead‑time increases turn‑off loss because the high‑side MOSFET sees the body‑diode recovery of the low‑side, raising switching loss. |
| 4 *|*How does the ESR of the output capacitor influence the control‑loop stability of a voltage‑mode regulator?* | ESR introduces a zero in the loop transfer function that can provide phase boost, aiding stability. However, if ESR is too low (e.g., all‑ceramic caps), the zero moves to high frequency, reducing phase margin; additional compensation is required. |
| 5 | *Why must the input filter of a switching regulator meet both conducted and radiated EMI limits, and how is the filter order chosen?* | Switching edges generate high‑frequency harmonics that can travel on power lines (conducted) and act as antennas (radiated). A π‑filter (LC‑C) or LCL provides attenuation at the switching frequency and its multiples. Order is chosen to achieve >20‑30 dB attenuation at the dominant harmonic while keeping component size reasonable. |
